# 30-Day Python Fundamentals Challenge
Level: Beginner -> Early Intermediate
Ziel: Python-Grundlagen festigen, selbstständig denken, GPT-Abhängigkeit reduzieren

## Über diese Challenge
Diese Challenge hilft mir, die **Grundlagen von Python zu meistern**, indem ich täglich kleine Aufgaben löse **Ohne GPT**
Ich darf:
- Dokumentationen lesen
- Googlen
- Eigene Fehler analysieren
- Debuggen
aber **keine KI verwenden**, um Lösungen zu schreiben oder Debugging zu übernehmen.

Das Ziel ist, dass ich **Selbstständig** werde, meine **Denkfähigkeit stärke** und die **Fundamente von Python** wirklich verinnerliche.

## Die 30-Tage-Challenge
**Die komplette Challenge findest du in der Datei: 30_days_challenge.md**

**Die struktur dieses Repositories**
python-30-days-challenge
-- day_01/
-- day_02/
-- day_03/
-- ...
-- day_30/
-- 30_days_challenge.md
-- README.md
-- LICENSE

Jeder Ordner enthält:
- Meine Lösungen
- Ein kurzes Lerntagebuch (max. 5 Zeilen)

## Tägliches Lerntagebuch (Template)
Füge in jeden Tages-Ordner eine datei "log.md" ein:

"# Day X - Log
   **Was habe ich gelernt?**
   -
   **Was war schwer?**
   -
   **Was hat mir besonders geholfen**
   -
   **Wie ist mein Gefühl heute?**
   -
   **GPT genutzt?**
   Nein / Ja (und warum)
"

- Python-grundlagen vollständig verstehen
